# CAP AI — Multi-County Shared Services Hub

Enterprise Streamlit platform for local government shared services, treasury oversight, fraud analytics, compliance monitoring, and AI transformation.

## Quick Start

```bash
cd cap_ai_multi_county_hub
pip install -r requirements.txt
streamlit run app.py
```

## Demo Login

- admin / admin123 (Administrator)
- manager / manager123 (County Manager)
- treasury / treasury123 (Treasury Officer)
- auditor / auditor123 (Auditor)
- viewer / viewer123 (Viewer)

## OpenAI Copilot

Set OPENAI_API_KEY for live AI responses. Demo mode works without it.
